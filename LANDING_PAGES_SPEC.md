## OVERVIEW & STRATEGY

### Purpose
Each challenge needs a dedicated landing page optimized for Meta Ads traffic. These pages should:
- Load fast (under 3 seconds)
- Have a single clear CTA (Buy Now)
- Address objections before they arise
- Build trust through social proof
- Create urgency without being pushy

### URL Structure
```
tradersyard.com/challenges/5k
tradersyard.com/challenges/25k
tradersyard.com/challenges/50k
tradersyard.com/challenges/100k
tradersyard.com/challenges/200k
tradersyard.com/gift/claim
tradersyard.com/flash-sale
tradersyard.com/reset
```

## DESIGN SYSTEM

### Color Palette
```
Primary Background: #0a0a0f (Deep black)
Secondary Background: #12121a (Card backgrounds)
Accent Green: #10b981 (Emerald - money, success)
Accent Gold: #f59e0b (Premium feel)
Text Primary: #f0f0f5 (White)
Text Secondary: #8888a0 (Gray)
Error/Urgency: #ef4444 (Red)
Success: #22c55e (Green)
```

### Typography
```
Headlines: Instrument Serif or Playfair Display (serif for sophistication)
Body: DM Sans or Inter (clean, readable)
Numbers/Data: JetBrains Mono (monospace for trading aesthetic)
```

### Key Design Elements
- Dark theme (trading terminal aesthetic)
- Subtle grid background pattern
- Green glow effects on CTAs
- Card-based sections with subtle borders
- Trust badges prominently displayed
- Mobile-first responsive design
